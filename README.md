# lunytales-website

Astro static site for the Luny Tales marketing website.

## Full technical documentation

- See `docs/PROJECT_HANDBOOK.md` for architecture, configuration, SEO/i18n rules, CI/CD, and migration notes.
- Spanish version: `docs/PROJECT_HANDBOOK.es.md`.

## Local development

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Build output is generated in `dist/` (never edit `dist/` manually).

## Deploy

- Production is served from Cloudflare Pages (`main` branch).
- Preview deployments are generated by Cloudflare Pages for non-main branches.
- `.github/workflows/deploy.yml` now runs CI checks only (build + smoke tests + SEO output checks).

## Key routes

- `/` (ES home)
- `/en/` (EN home)
- `/privacy/`
- `/terms/`
- `/faq/` (ES FAQ)
- `/en/faq/` (EN FAQ)

## Configuration

Central site config lives in `src/config/site.ts`:

- `SITE_CONFIG.origin` (single source of truth for canonical origin)
- `IS_STAGING` from environment flags (`PUBLIC_IS_STAGING` and preview context)
- `hotmartUrl`
- `paths` (home, legal pages, demo, OG images)
- `trackingGate`

SEO helpers are in `src/lib/seo.ts`.

## Environment behavior

- Production builds keep `meta robots` as `index,follow` and `robots.txt` as `Allow: /`.
- Preview/staging builds enforce `meta robots` as `noindex,nofollow` and `robots.txt` as `Disallow: /`.
- `trackingGate: true` keeps tracking disabled unless URL contains `?track=1`.
